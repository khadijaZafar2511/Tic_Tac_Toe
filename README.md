# Interactive Tic-Tac-Toe Game

A lightweight, responsive, and interactive **Tic-Tac-Toe game** built entirely with vanilla web technologies. Play smoothly directly inside your web browser.



## 🚀 Features

* **Pure Web Tech:** Zero dependencies, heavy libraries, or complex build tools.
* **Smart Game Logic:** Dynamic win detection and draw validation using JavaScript arrays.
* **Click Protection:** Prevention of double-clicking or overwriting occupied squares.
* **Instant Reset:** Quick restart button to clear the board grid instantly.
* **Responsive Layout:** Flexbox/Grid styling that scales perfectly on mobile and desktop screens.

## 🛠️ Built With

* **HTML5:** Semantic structure for the game board and control interface.
* **CSS3:** Modern layouts, grid lines, and interactive hover effects.
* **JavaScript (ES6):** Turn-based state handling, DOM manipulation, and winning combination checkers.

## 🎮 How to Play Locally

You do not need to install any servers or packages to run this application.

### Setup Steps
1. Clone this repository to your machine:
   ```bash
   git clone https://github.com
   ```
2. Navigate into the folder:
   ```bash
   cd YOUR_REPOSITORY_NAME
   ```
3. Open the game:
   * Double-click the `index.html` file to launch it in your default web browser.

## 🕹️ Game Rules

1. The game is played on a classic 3x3 grid.
2. Player 1 is `X` and Player 2 is `O`. Players take turns clicking empty squares.
3. The first player to get 3 of their marks in a row (up, down, across, or diagonally) wins.
4. If all 9 squares are full and no player has 3 in a row, the game ends in a tie.

## 🧠 Key Learning Points

* **DOM Manipulation:** Selecting, changing, and updating HTML elements dynamically via JavaScript events.
* **State Management:** Tracking current player turns, board data arrays, and game active statuses.
* **Condition Checking:** Iterating through multi-dimensional winning index arrays to find a match.
